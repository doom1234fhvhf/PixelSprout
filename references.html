Special Thanks & Learning Resources

While all code and visuals were independently created by students, the following educational resources helped guide best practices in web design, accessibility, and development structure.

  • MDN Web Docs — Reference for HTML semantics, CSS properties, and JavaScript behavior.
  • W3C Web Accessibility Initiative (WAI) — Guidance on accessible markup, contrast, and keyboard navigation.
  • Google Web Dev Resources — Inspiration for performance, layout organization, and responsive design.
  • CSS Tricks (educational articles) — Conceptual understanding of layout techniques like Flexbox and Grid.
  • Technology Student Association (TSA) — Competition guidelines and project expectations.

No templates, frameworks, or prebuilt components were copied or reused. All implementation decisions were made by the student team.
